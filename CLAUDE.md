## Project Overview

**Lantern（鱼灯展）** — 鱼灯非遗文化数字化宣传网站。以"河流叙事"为视觉线索，通过 3D 交互模型、滚动动效、地缘文旅地图，让观众在浏览器中完成"了解鱼灯 → 拆解工艺 → 查询实地观赏地点"的完整体验。

## Environment

- **OS**: Windows 11 (win32)
- **Shell**: bash (Git Bash)
- **No build system, package manager, or test framework** — all dependencies loaded via CDN

## Tech Stack & Conventions

### 页面搭建

- **原生 HTML + CSS + JavaScript**，不引入 React / Vue 等前端框架，保持轻量
- **Three.js** 通过 CDN 引入作为 3D 渲染核心
- **GSAP + ScrollTrigger** 通过 CDN 引入，驱动滚动动画
- **CSS 变量** 统一管理两套色彩体系：
  - 暗黑底色（模拟鱼灯夜游场景）
  - 暖金色流光（鱼灯标志性色调，页面中唯一的亮色系）
- **响应式布局** 覆盖手机、平板、桌面端，3D 模型和 UI 元素在不同屏幕下自适应缩放

### 3D 模型与交互

- 鱼灯模型使用 **`.glb` 格式**，通过 **Draco 压缩** 将体积降至原始的 10%–20%
- 交互依赖 Three.js 的 **OrbitControls**（旋转/缩放）和 **Raycaster**（点击部位检测）
- 点击模型不同部位 → 弹出对应工艺细节（扎骨、裱糊、绘彩、灯芯）
- 入场上使用"生长动画"：骨架自下而上逐层拼接，表面泛起暖金色光泽
- 配合 **粒子系统** 模拟夏夜萤火与火星效果

### 动效与叙事

#### 开场动画——河流、徽派建筑与鱼灯

开场为一幅 **夜间横屏画卷**：夜幕如墨，**一条河流横贯画面**，两岸是错落的徽派建筑剪影（粉墙黛瓦、马头墙），窗户透出暖黄灯火。河面倒映月光与灯火，泛着粼粼波光。一只鱼灯在河流中央上方，围绕居中的 **"开始"字样做非匀速转动**：

- 转动节奏模拟 **鱼在水中跳跃、扑腾后落回水中的加速度**——忽快忽慢，带弹性缓动（easing），仿佛鱼灯活了过来
- 轨迹并非正圆，而是略带 **椭圆 + 上下浮动**，增强鲜活感
- 鱼灯尾部拖曳暖金色流光，划过夜空
- 整体氛围 **静谧、深邃**，鱼灯的金色在暗夜中格外夺目

两岸徽派建筑细节：
- 近处建筑层次分明，远处淡化成剪影
- 窗户透出暖黄灯火，与鱼灯的暖金色呼应
- 河面倒影随鱼灯游动产生涟漪扭曲

#### 点击"开始"——鱼灯入河，三板块展开

用户点击"开始"字样后：

1. **鱼灯加速旋转半圈**，然后以一个漂亮的弧线 **朝观者方向游来（游向屏幕外）**，身体在视野中迅速放大，最终带着一道暖金色流光冲出画面
2. 同时 **镜头缓缓推进，河流逐渐放大**，河面占据更多画面
3. 从河面下 / 画面两侧 **升起另外两条鱼灯**（颜色/形态略有区分），与原鱼灯形成 **三条鱼灯** 的格局
4. **三条鱼灯分别代表网站的三大内容板块**，各自游向河面的上、中、下游区域并停留
5. 三鱼灯呈倒品字或沿河流分段排布，各自伴随板块标题淡入
6. 整体过渡使用 GSAP 时间线编排，确保每个动作的缓动曲线和时长精准衔接

> **设计意图**："一条鱼 → 三条鱼"的转变隐喻"一个人点亮一盏鱼灯 → 鱼灯文化繁衍扩散"。非匀速转动让鱼灯有生命感而非机械钟表，徽派建筑作为徽州地域的文化锚点，将鱼灯扎根在具体的地理语境中。

#### 滚动叙事

- **SVG 曲线** 作为贯穿全页的"河流"背景层，滚动时一笔一划延伸
- GSAP ScrollTrigger 驱动内容板块沿河流方向错落淡入
- 三板块内容在三条鱼灯的引导下依次展现

### 性能规范

- 图片统一使用 **WebP** 格式
- 视频使用 **WebM** 编码
- 非屏幕可见区域的资源（图片、3D 资产）全部 **懒加载**
- 3D 模型必须经过 Draco 压缩后再上线

### 地缘文旅地图

- 数据层使用 **GeoJSON** 存储地点坐标与简介
- 地图渲染使用支持自定义风格的矢量地图组件
- 配色与整站暗金主题一致，缩放级别对应不同信息密度
- 点击标记 → 侧边弹出信息面板（鱼灯特色、观赏时段、交通概况）
